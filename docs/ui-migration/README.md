### **项目启动时**
- 通读整个文档了解项目全貌
- 重点关注"改造原则"和"不可变更清单"
- 理解"换皮不换骨"的核心理念

- 根据进度跟踪表了解当前状态
- 按照阶段3实施计划执行具体任务
- 对照验收标准检查完成质量

- 查看风险评估部分寻找解决方案
- 参考"紧急回滚方案"保证项目稳定性
